# Local AI Integration

Integrate local AI models (Ollama) into VS Code with project context.

## Features

- **Project Context**: Automatically include the active file in your AI queries.
- **Multi-File Patching**: Apply AI-suggested changes directly to any project file with a visual diff.
- **Cinematic Background**: Immersive "Black Hole" theme for a premium coding experience.
- **Real-time Streaming**: See AI responses word-by-word as they are generated.

## Requirements

- [Ollama](https://ollama.com/) must be installed and running on your machine.

## Usage

1. Open the **Local AI** view in the sidebar.
2. Select your model.
3. Start chatting!
